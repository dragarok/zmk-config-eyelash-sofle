# Eyelash Sofle вҖ” Keymap Reference

A human-readable cheat sheet for this keyboard. The source of truth is
[`config/base.keymap`](config/base.keymap); this doc is generated by hand from it.

Layout style: urob-style 5-column home-row-mod layout, mapped onto the eyelash
sofle's larger physical board (number row + center arrow column + encoder +
extra thumb keys). Legend: `вҢҳ`=GUI/Cmd, `вҢҘ`=Alt, `вҢғ`=Ctrl, `вҮ§`=Shift.

---

## Layers at a glance

| # | Layer | How you reach it |
|---|-------|------------------|
| 0 | **Base** | default |
| 1 | **Nav** | **hold** the Space thumb |
| 2 | **Fn** | **hold** the Return/Fn thumb |
| 3 | **Num** | **hold** `G`, or use the smart-num thumb |
| 4 | **Sys** | **hold Fn + Num together** (auto-activates) |
| 5 | **Mouse** | **hold** the inner-left Mouse/Return thumb |

## The constant frame (same on every layer)

These keys don't change between layers:

- **Number row:** `` ` `` `1 2 3 4 5` В· `6 7 8 9 0` В· `Bksp`
- **Left outer column:** `Tab` В· `Esc` В· (bottom mod) вҖ” each is a home-row-mod (see below)

## Layer 1 вҖ” Nav (hold Space)

```
  В·      В·    вҮ§Tab  Cmd-Tab LClick в”Ӯ  вҶ“вҶ’     вҶ’    PgUp   PgDn    В·
 вҢҳsk    вҮ§sk   вҢҘsk   вҢғsk    RClick в”Ӯ  вҶҗ     Bksp   Tab    Del    Ret
  В·      В·      В·      В·    MClick в”Ӯ  вҶ‘      В·     Ins     В·      В·
                    [ ]  [ ]       в”Ӯ  [ ]  [Cancel]
```

- Left hand = **sticky mods** (`вҢҳ вҮ§ вҢҘ вҢғ`) + mouse clicks. Tap a sticky mod, then a key.
- `Cmd-Tab` = **app switcher** that stays open while you keep tapping (tri-state swapper).
- Right hand = **arrow/nav cluster** with smart long-presses:
  - `вҶҗ`/`вҶ’` long-press = **start / end of line**
  - `вҶ‘`/`вҶ“` long-press = **start / end of document**
  - `Bksp`/`Del` long-press = **delete word** backward / forward
- `Cancel` = stop caps-word / num-word.

## Layer 4 вҖ” Sys (hold Fn + Num together)

```
        BT0 BT1 BT2 BT3 BTclr     в”Ӯ   В·      В·      В·      В·      В·
RGB-eff RGB-efr RGB+  RGB-  Boot   в”Ӯ  Boot    В·      В·      В·      В·
```

- **Bluetooth:** `BT0вҖ“BT3` select profile, `BTclr` clears the current profile.
- **RGB underglow:** on/off, effect, brightness Вұ, speed.

## Smart-key cheat sheet (the hard-to-remember stuff)

| Key | Tap | Double-tap / Shift-tap | Hold |
|-----|-----|------------------------|------|
| **smart-shift** (R thumb) | sticky вҮ§ | **Caps-Word** | вҮ§ |
| **smart-num** (R thumb) | **Num-Word** (auto-number mode) | sticky Num layer | Num layer |
| **Space** (L thumb) | space | `. ` + auto-capitalize next | Nav layer |
| **Bksp** | backspace | вҮ§+tap = Delete | вҖ” |
| **copy/cut** | copy | cut | вҖ” |
| **Hyper** thumb | desktop manager | вҖ” | Hyper (вҢғвҮ§вҢҘвҢҳ) |

**Word-modes:** *Caps-Word* keeps Shift on until you type a non-letter (great for
`CONSTANT_NAMES`). *Num-Word* keeps the Num layer on for one number, then drops back.

**Outer-column / bottom home-row mods:** `Tab` (hold = вҢҘвҢҳ), `Esc` (hold = вҢғвҮ§вҢҳ),
and the bottom corner keys also hold extra mod combos вҖ” see `base.keymap` lines
160вҖ“164 if you want the exact chords.
