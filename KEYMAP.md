# Eyelash Sofle — Keymap Reference

A human-readable cheat sheet for this keyboard. The source of truth is
[`config/base.keymap`](config/base.keymap); this doc is generated by hand from it.

> **Rendered diagram:** ![keymap](keymap-drawer/eyelash_sofle.svg)
> Auto-generated by CI on every keymap change — see [`keymap-drawer/eyelash_sofle.svg`](keymap-drawer/eyelash_sofle.svg).

Layout style: urob-style 5-column home-row-mod layout, mapped onto the eyelash
sofle's larger physical board (number row + center arrow column + encoder +
extra thumb keys). Legend: `⌘`=GUI/Cmd, `⌥`=Alt, `⌃`=Ctrl, `⇧`=Shift.

---

## Layers at a glance

| # | Layer | How you reach it |
|---|-------|------------------|
| 0 | **Base** | default |
| 1 | **Nav** | **hold** the Space thumb |
| 2 | **Fn** | **hold** the Return/Fn thumb |
| 3 | **Num** | **hold** `G`, or use the smart-num thumb |
| 4 | **Sys** | **hold Fn + Num together** (auto-activates) |
| 5 | **Mouse** | **hold** the inner-left Mouse/Return thumb |

---

## The constant frame (same on every layer)

These keys don't change between layers:

- **Number row:** `` ` `` `1 2 3 4 5` · `6 7 8 9 0` · `Bksp`
- **Left outer column:** `Tab` · `Esc` · (bottom mod) — each is a home-row-mod (see below)
- **Right outer column:** `\` · `'` · `Return`
- **Center column:** `↑ ↓ ← →` (becomes **mouse-move** on the Nav & Fn layers)
- **Encoder:** rotate = **Volume Up / Down**, press = **Mute**
- **Thumb mods (left):** `Ctrl` · `Gui` and the layer thumbs below

## Layer 0 — Base (QWERTY + home-row mods)

```
  Q      W      E      R      T   │   Y      U      I      O      P
 A/⌘    S/⇧    D/⌥    F/⌃   G/Num │   H     J/⌃    K/⌥    L/⇧    ;/⌘
  Z      X      C      V      B   │   N      M      ,      .      /
                  [Fn] [Space]    │   [Num] [⇧smart]
```

- **Home-row mods** — hold for the mod, tap for the letter:
  `A`=⌘ `S`=⇧ `D`=⌥ `F`=⌃  |  `J`=⌃ `K`=⌥ `L`=⇧ `;`=⌘
- `G` — tap `g`, **hold = Num layer**
- Left thumbs: **Fn** (tap = Return) · **Space** (tap = space, hold = Nav)
- Right thumbs: **smart-num** · **smart-shift** (see cheat sheet)

## Layer 1 — Nav (hold Space)

```
  ·      ·    ⇧Tab  Cmd-Tab LClick │  ↓→     →    PgUp   PgDn    ·
 ⌘sk    ⇧sk   ⌥sk   ⌃sk    RClick │  ←     Bksp   Tab    Del    Ret
  ·      ·      ·      ·    MClick │  ↑      ·     Ins     ·      ·
                    [ ]  [ ]       │  [ ]  [Cancel]
```

- Left hand = **sticky mods** (`⌘ ⇧ ⌥ ⌃`) + mouse clicks. Tap a sticky mod, then a key.
- `Cmd-Tab` = **app switcher** that stays open while you keep tapping (tri-state swapper).
- Right hand = **arrow/nav cluster** with smart long-presses:
  - `←`/`→` long-press = **start / end of line**
  - `↑`/`↓` long-press = **start / end of document**
  - `Bksp`/`Del` long-press = **delete word** backward / forward
- `Cancel` = stop caps-word / num-word.

## Layer 2 — Fn (hold Return/Fn)

```
 F12    F7     F8     F9     ·    │   ·     Prev  Vol+   Next    ·
F11/⌘  F4/⇧   F5/⌥   F6/⌃    ·    │   ·    Dsk←  Vol-   Dsk→     ·
 F10    F1     F2     F3     ·    │  PinApp PinWin DskMgr  ·      ·
                    [ ]  [ ]      │  [Mute] [Play/Pause]
```

- Left = **F-keys** (with the same home-row mods on the middle row).
- Right = **media** (Prev / Next / Vol± / Play-Pause / Mute) and **virtual-desktop**
  control: `Dsk←`/`Dsk→` switch desktops, `PinWin`/`PinApp` pin across desktops,
  `DskMgr` opens the desktop manager.

---

## Layer 3 — Num (hold G, or smart-num thumb)

```
  ·      7      8      9      ·    │   !      (      )      {      }
 0/⌘    4/⇧    5/⌥    6/⌃     ·    │   *      -      _      [      ]
  ·      1      2      3      ·    │   #      =      +      ^      &
                    [ ]  [ ]       │  [ ] [ ]
```

- Left = **numpad** (`7 8 9 / 4 5 6 / 1 2 3`, with `0` on the pinky). Home-row mods
  still active on the middle row.
- Right = **symbols**: `! ( ) { } * - _ [ ] # = + ^ &`.

---

## Layer 4 — Sys (hold Fn + Num together)

```
        BT0 BT1 BT2 BT3 BTclr     │   ·      ·      ·      ·      ·
RGB-eff RGB-efr RGB+  RGB-  Boot   │  Boot    ·      ·      ·      ·
RGB-off RGB-on  RGB-spd  ·  Reset  │  Reset   ·      ·      ·      ·
```

- **Bluetooth:** `BT0–BT3` select profile, `BTclr` clears the current profile.
- **RGB underglow:** on/off, effect, brightness ±, speed.
- **Boot / Reset:** `Boot` = bootloader (for flashing), `Reset` = soft reset.

---

## Layer 5 — Mouse (hold inner-left Mouse thumb)

```
  ·      ·      ·      ·      ·    │  M↓     M→    PgUp   PgDn    ·
  ·      ·      ·      ·      ·    │  M←   LClick MClick RClick   ·
  ·      ·      ·      ·      ·    │  M↑   Wh←    Wh↑    Wh↓    Wh→
```

- Right hand = mouse **move** (`M↑↓←→`), **clicks** (L/M/R), and **wheel** (`Wh`).

## Smart-key cheat sheet (the hard-to-remember stuff)

| Key | Tap | Double-tap / Shift-tap | Hold |
|-----|-----|------------------------|------|
| **smart-shift** (R thumb) | sticky ⇧ | **Caps-Word** | ⇧ |
| **smart-num** (R thumb) | **Num-Word** (auto-number mode) | sticky Num layer | Num layer |
| **Space** (L thumb) | space | `. ` + auto-capitalize next | Nav layer |
| **Bksp** | backspace | ⇧+tap = Delete | — |
| **copy/cut** | copy | cut | — |
| **Hyper** thumb | desktop manager | — | Hyper (⌃⇧⌥⌘) |

**Word-modes:** *Caps-Word* keeps Shift on until you type a non-letter (great for
`CONSTANT_NAMES`). *Num-Word* keeps the Num layer on for one number, then drops back.

**Outer-column / bottom home-row mods:** `Tab` (hold = ⌥⌘), `Esc` (hold = ⌃⇧⌘),
and the bottom corner keys also hold extra mod combos — see `base.keymap` lines
160–164 if you want the exact chords.
